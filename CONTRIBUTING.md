# Panduan Berkontribusi (Contributing Guidelines)

Terima kasih telah mempertimbangkan untuk berkontribusi! 

Semua kontribusi, laporan bug, perbaikan bug, peningkatan dokumentasi, peningkatan, dan ide-ide sangat diterima.

Langkah untuk berkontribusi:
1. Fork repositori
2. Buat branch baru (`git checkout -b fitur-saya`)
3. Lakukan commit (`git commit -am 'Menambahkan fitur'`)
4. Push ke branch (`git push origin fitur-saya`)
5. Buat Pull Request baru
